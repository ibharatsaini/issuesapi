## Installation


```
  cd issuesapi
  npm install
  
```
    
## Run Locally

Clone the project

```cli
  git clone  https://github.com/ibharatsaini/issuesapi.git
```

Go to the project directory

```cli
  cd issuesapi
```

Install dependencies

```cli
  npm install
```

Start the server

```cli
  npm start
```

Go to  http://localhost:3000 .
